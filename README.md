## Git commands
- `git init` - Initialize a new git repository
- `git add <file>` - Add a file to the staging area
- `git commit -m "message"` - Commit changes with a message
- `git status` - Check the status of the repository
- `git log` - View commit history
- `git branch` - List branches
- `git checkout <branch>` - Switch to a different branch
- `git merge <branch>` - Merge a branch into the current branch
- `git remote add origin <url>` - Add a remote repository
- `git push -u origin <branch>` - Push changes to a remote repository
- `git pull` - Fetch and merge changes from a remote repository
- `git clone <url>` - Clone a remote repository
- `git diff` - Show changes between commits, branches, or working directory
- `git reset --hard <commit>` - Reset the current branch to a specific commit
- `git stash` - Stash changes in the working directory
- `git tag <tagname>` - Create a new tag
- `git show <commit/tag>` - Show details of a specific commit or tag
- `git revert <commit>` - Revert a specific commit
- `git fetch` - Fetch changes from a remote repository without merging
- `git rebase <branch>` - Reapply commits on top of another base branch
- `git cherry-pick <commit>` - Apply the changes from a specific commit onto the
current branch
- `git log --oneline` - View commit history in a condensed format
- `git log --graph` - View commit history as a graph
- `git log --author="name"` - View commits by a specific author
- `git log --since="date"` - View commits since a specific date
- `git log --until="date"` - View commits until a specific date
- `git log --grep="message"` - View commits with a specific message
- `git log --stat` - View commit history with file changes statistics
- `git log --patch` - View commit history with diffs
- `git log --name-only` - View commit history with changed file names
- `git log --name-status` - View commit history with changed file names and status
- `git log --pretty=format:"%h - %an, %ar : %s"` - View commit history with a custom format
- `git log --follow <file>` - View commit history for a specific file, including renames
- `git log --all` - View commit history for all branches
